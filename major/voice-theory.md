# 钢琴
> 88键:
> (`A2,B2`/`C1,D1,E1,F1,G1,A1,B1`/`C,D,E,F,G,A,B`)
> (`c,d,e,f,g,a,b`/`c1,d1,e1,f1,g1,a1,b1`/`c2,d2,e2,f2,g2,a2,b2`/)
> (`c3,d3,e3,f3,g3,a3,b3`/`c4,d4,e4,f4,g4,a4,b4`/5)
> 低音区(大二组，大一组，大字级),
> 中音区(小字组，小一组，小二组),
> 高音区(小三组，小四组，小五组)
> c1为中央C音，a1为国际标准A音

# 音符
> 相邻元间隔两键关系为半音
> 两键间隔一个音关系为全音

> 休止符0
> 延音线: 多个相同音高的音连一起变成一个音
> 连音线: 音与单之间元间隙，但音高变化
> 减时线: 下划线数量表示除2次数

> 2+3分组(5黑+7白=12平均律), 音阶从左往右上升
> 每组的白键音名：`CDEFGAB`
> 升降音名(还原号可以恢复)：升半音白键`#`，降半音白键`b`
> 重升(`x`)重降(`bb`)升降两个半音

> n分音符(1/n全音符)
> 时值(单位beat): 四分音符为1beat
> BPM: beats per minutes
> BPS: BPM/60s

> x/y拍(每小节x拍, 以y拍为1拍, 容量x*y)
> 附点音符(音符加音符的一半,1.5倍音符)
> 三连音: 总时值不变，内部3等分，标记数字"3"

# 名
> 唱名/简谱(do1   re2   mi3   fa4   sol5  la6   si7)
> 乐名     (主音 上主音 中音 下属音 属音 下中音 导音)
> 音名     (C     D     E     F      G     A     B)

# 调式
| 调式      | 音程结构(W全H半)   | 色彩      |
| --------  | ------------- | ------- |
| 自然大调  | W-W-H-W-W-W-H | 明亮、稳定   |
| 自然小调  | W-H-W-W-H-W-W | 暗淡、忧伤   |
| 多利亚    | W-H-W-W-W-H-W | 神秘、爵士   |
| 弗里几亚  | H-W-W-W-H-W-W | 西班牙、重金属 |
| 利底亚    | W-W-W-H-W-W-H | 梦幻、飘浮感  |


> 音程: 还原后有几个白键几度
> 标准大音程: 自然大调音阶中各音到主音的距离(半音数: 0,2,4,5,7,9,11,12)
> 前缀: (1458纯),(2367,间隔半音在(对应标准大)则大,在(对应标准大-1)则小,其他用增减)

```markdown
给定：度数 n，半音数 h

IF n ∈ {1, 4, 5, 8}:
    标准值 = [0, 5, 7, 12][对应位置]
    IF h == 标准值: "纯n度"
    IF h < 标准值: "减n度"（每少1半音多一个"减"）
    IF h > 标准值: "增n度"（每多1半音多一个"增"）

IF n ∈ {2, 3, 6, 7}:
    标准大 = [2, 4, 9, 11][对应位置]
    IF h == 标准大: "大n度"
    IF h == 标准大 - 1: "小n度"
    IF h < 标准大 - 1: "减n度"
    IF h > 标准大: "增n度"
```

# 和弦
> 三个及三个以上不同音构成
> 本调和弦: 没有升降
> 离调和弦: 有升降


* 三和弦
> 三个音(根音、三音、五音)按三度关系叠加的和弦(大三和弦、小三和弦、减三和弦、增三和弦)
> 根音(最低音，原位和弦)
> 三音(根音上方三度音)
> 五音(根音上方五度音)

| 类型     | 三度组合        | 半音数| 标记 |
| ----     | --------------- | ----- | ---- |
| 大三和弦 | 大三度 + 小三度 | 4+3=7 | C    |
| 小三和弦 | 小三度 + 大三度 | 3+4=7 | Cm   |
| 增三和弦 | 大三度 + 大三度 | 4+4=8 | Caug |
| 减三和弦 | 小三度 + 小三度 | 3+3=6 | Cdim |


* 七和弦
> 四个音(根音、三音、五音、七音)按三度关系叠加的和弦

| 七和弦类型| 三度组合 | 半音计算 | 标记  |
| --------- | -------- | -------- | ----- |
| 大七和弦  | 大+小+大 | 4+3+4=11 | CMaj7 |
| 属七和弦  | 大+小+小 | 4+3+3=10 | C7    |
| 小七和弦  | 小+大+小 | 3+4+3=10 | Cm7   |
| 半减七和弦| 小+小+大 | 3+3+4=10 | Cm7♭5 |
| 减七和弦  | 小+小+小 | 3+3+3=9  | Cdim7 |


* 九和弦
> 五个音(根音、三音、五音、九音)按三度关系叠加的和弦
> ADD9(三和弦+九音)

| 九和弦类型| 构成          | 九音性质       | 标记  |
| --------- | ------------- | -------------- | ----- |
| 大九和弦  | Maj7 + 大三度 | 大九度（14→2） | CMaj9 |
| 属九和弦  | 7 + 大三度    | 大九度（14→2） | C9    |
| 小九和弦  | m7 + 大三度   | 大九度（14→2） | Cm9   |


* 挂留和弦
> 三和弦的三音被"挂"起来（换成二度或四度），悬空不定，需要解决

* 转位和弦
> 转位和弦 = 同样的音，换不同的低音，产生不同的音响层次和功能
